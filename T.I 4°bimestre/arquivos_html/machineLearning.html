Machine Learning

Machine Learning é uma área da Inteligência Artificial que se concentra em usar dados e algoritmos para imitar a maneira como os humanos aprendem, melhorando gradualmente sua precisão1. Machine Learning pode ser usado para analisar uma grande quantidade de dados e promover o reconhecimento de padrões, fazer previsões e tomar decisões com o mínimo de intervenção humana2. Machine Learning pode ser aplicado em vários domínios, como processamento de linguagem natural, visão computacional, reconhecimento de voz, jogos, robótica, etc
